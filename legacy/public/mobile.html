Application mobile citoyenne & député — React Native / Expo

Déclinaison mobile-first du design system du portail web : mêmes jetons de couleur, mêmes composants (boutons, cartes, badges de statut, barre de progression législative), pensée pour un usage terrain — connexion lente, plein soleil (contrastes AA), zones tactiles ≥ 44 px, bilingue FR/AR avec miroir RTL complet.

Mini design system partagé web + mobile

Jetons exportables tels quels vers React Native (StyleSheet / thème). L'or n'est jamais utilisé pour du texte long (contraste) : la variante « or texte » sert aux libellés courts sur fond clair. Le rouge est réservé aux alertes et à l'urgence nationale.
